# wiki2md

CLI to fetch a MediaWiki page via API and save it as Markdown.

## Install

```bash
# From a local checkout
pipx install .
# or
pip install -e .

## Usage
wiki2md --url "https://forgottenrealms.fandom.com/wiki/Menhir" --outdir . --fix-fandom-images -f Menhir.md
```
